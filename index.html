El proceso

Este es el proceso que seguirán los seleccionados.

  • Dia 1/ Sesión 1

    HTML

    Aprenderemos a construir la estructura basica de nuestra desarrollando un proyecto real!

  • Dia 1/ Sesión 2

    CSS

    Aprenderemos a darle vida a nuestro proyecto con selectores y estilos de CSS.

  • Dia 2

    Javascript

    En este dia aprenderemos todas las bases de este lenguaje de programación y le daremos dinamismo a nuestro proyecto.

  • 4 Semanas

    Mentorias

¿ Que aprenderas ?

El contenido del programa

  • HTML
  • 1. Aprenderas sobre <!DOCTYPE html><html> <head> <body>
  • 2. Aprenderas sobre las etiquetas usadas dentro del <head>
  • 3. Aprenderas sobre las etiquetas usadas dentro del <body>
  • 4. Aprenderas los atributos id y class
  • 5. Aprenderas sobre el uso de Formularios.
  • CSS
  • 1. Aprenderas sobre los estilos en cascada y la especificidad.
  • 2. Aprenderas sobre los selectores en CSS.
  • 3. Aprenderas sobre las seudoclases en CSS.
  • 4. Aprenderas sobre el modelo de caja.
  • 5. Aprenderas a agregar Fuentes e Iconos.
  • 6. Aprenderas sobre Layouting con CSS.
  • 7. Aprenderas sobre media queries y Responsive Design.
  • JavaScript
  • 1. Aprenderas a manipular eventos de mouse onclick, onmouseover
  • 2. Aprenderas a manipular eventos de teclado onkeypress, onkeydown y onkeyup.
  • 3. Aprenderas sobre eventos de documentos onload y onunload.
  • 4. Aprenderas sobre preventDefault(), stopPropagation(), addEventListener().
  • 5. Aprenderas sobre manejo de Scope, Arrays, Objetos, Condicionales, Funciones.
  • 6. Aprenderas sobre manipulación del DOM.
  • 7. Aprenderas sobre Web Storage API.
  • 8. Aprenderas sobre Fetch API.
